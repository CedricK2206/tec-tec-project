# Tec-Tec Cream — Project Files

This repository is the single source of truth for the Tec-Tec Cream sunscreen launch project.

Tec-Tec Cream is a reef-safe mineral sunscreen launching in Mauritius, named in memory of Xavier ("Tec-Tec"), with a fixed donation per tube sold going to a Mauritian skin cancer cause partner.

## Files

| File | Description |
|------|-------------|
| `TEC_TEC_MEMORY.md` | Master project memory — decisions, open questions, contacts, risks, next actions |
| `BRAND.md` | Brand positioning, tone, visual identity direction |
| `COMPETITORS.md` | Competitor intelligence and market landscape |
| `REGULATORY.md` | Mauritius regulatory & compliance notes |
| `SUPPLIERS.md` | Supplier & contact log |
| `FINANCIALS.md` | Unit economics, pricing, margin structure |

## Project Status

**Phase 1 — Foundations** | Timeline: six months to launch

## How to Use

The memory file is updated at the end of each working session. All decisions are dated. Superseded decisions are retained for audit trail, not deleted.
